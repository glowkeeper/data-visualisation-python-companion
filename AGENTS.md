# DVI — Data Visualisation — persistent project brief

## Purpose

DVI is a University of Roehampton Data Visualisation teaching module organised around **Data → Insight → Decision**. Tableau remains central. Python/Jupyter companion material should reveal analytical ideas that Tableau hides, not become a separate programming course.

## Teaching principles

- Notebooks are analytical conversations rather than Python tutorials.
- Markdown/Jupytext is the notebook source of truth; generated `.ipynb` files are run in a browser or VS Code with the project `.venv` selected.
- Weeks 1 and 2 explain what Tableau does; Week 3 begins explaining what analysts do.
- Week 3 is **Dashboard Storytelling** and should use fewer new Python concepts and more reflective “Think” moments.
- A dashboard answers a question and communicates a narrative; charts provide evidence, while the story provides meaning.
